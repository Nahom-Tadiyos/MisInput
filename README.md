

# 🎮 MisInput

> **A fast-paced typing survival game where your own keyboard turns against you.**
>
> Built with **Python** and **Raylib**, MisInput challenges players to type random phrases before time runs out. Every completed phrase breaks the keyboard by swapping random letter keys, forcing players to adapt as muscle memory becomes unreliable.

# 🚀 Installation

## Prerequisites

- Python 3.8+
- Git

---

## Clone the Repository

```bash
git clone https://github.com/Nahom-Tadiyos/MisInput

cd MisInput

pip install raylib
```


## Run the Game

```bash
python main.py
```


Every successful phrase makes the next one more difficult—not because the words get harder, but because your keyboard becomes increasingly unpredictable.
